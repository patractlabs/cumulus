The AuRa consensus algoritm for parachains.

This extends the Substrate provided AuRa consensus implementation to make it compatible for parachains. The main entry points for of this consensus algorithm are build_aura_consensus and [import_queue].

For more information about AuRa, the Substrate crate should be checked.
